## How to View

Simply open `index.html` in your web browser to view the portfolio.

For development with live reload, you can use any local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.

## Customization

To customize the portfolio for your needs:

1. Update personal information in `index.html`
2. Modify color scheme in `style.css` (CSS variables at the top)
3. Add/remove projects and skills as needed
4. Update contact information and links
